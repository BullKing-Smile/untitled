# Algorithm


## 二进制

负数表示方式
**负数-->正数二进制--> 减1 --> 取反**
<br/>
eg: 
 -6 --> 正数6二进制：0110 --> 减1 --> 0101 --> 取反 --> 1010
 -8 --> 正数8二进制：1000 --> 减1 --> 0111 --> 取反 --> 1000



负数二进制 值的计算方式

**二进制-->取反-->加1-->求值-->带上负号**
eg: 
1**** 1010 --> 取反 0**** 0101 --> 加1 0**** 0110 --> 算值 6 --> 加符号 -6



### 位移运算
**负数 >>, >>> 效果不一样**
负数>>2 左侧空缺 用符号位(1)来填
负数>>>2 交 带符号 右移， 左侧空缺用0来填

### 打印二进制
```java
public class BinarySystem {
    /**
     * 打印 二进制
     * int类型是是32位， 因此如果要计算long类型则需要用1L进行位移计算
     * 并且只能用==0计算， 因为是“与”运算
     * @param number 需要打印二进制的数字
     */
    public static void printBinary(int number) {
        StringBuilder sb = new StringBuilder();
        for (int i = 31; i >= 0; i--) {
            sb.append((number & (1<<i)) ==0 ? "0" : "1");
        }
        System.out.println(sb);
    }
}
```
